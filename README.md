Реализация расширенного алгоритма Евклида на языке **Golang**

*Расширенный алгоритм Евклида* — это метод вычисления не только наибольшего общего делителя (НОД) двух целых чисел \(a\) и \(b\), но и коэффициентов \(x\) и \(y\) для уравнения Безу ax+by=gcd(a,b). Этот алгоритм находит коэффициенты, которые представляют НОД как линейную комбинацию исходных чисел, что делает его незаменимым в таких областях, как криптография, например, для вычисления модульных обратных элементов.
